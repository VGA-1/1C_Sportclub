# Система управления фитнес-клубом «Клуб К-Атлет»

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Status](https://img.shields.io/badge/status-in%20development-orange)

## О проекте

Данный проект представляет собой комплексную информационную систему для автоматизации бизнес-процессов современного фитнес-клуба «Клуб К-Атлет». Система предназначена для учета клиентов (спортсменов), тренеров, абонементов, расписания, продаж товаров и финансовой отчетности.

Цель проекта — повышение эффективности управления клубом за счет цифровизации ключевых операций, предоставления сотрудникам удобного инструмента для работы и формирования детальной аналитики для владельцев.

## Функциональные возможности

### 1. Учет спортсменов и тренеров
- **Спортсмены:** Ведение полной базы данных спортсменов (личные данные, контакты, документы, законные представители, возрастные группы).
- **Тренеры:** Учет тренерского состава с указанием оказываемых услуг и закрепленных кабинетов.

### 2. Управление услугами и абонементами
- **Услуги:** Каталог предоставляемых клубом услуг (тренажерный зал, групповые занятия и т.д.).
- **Абонементы:** Гибкая система настройки видов абонементов (срок действия, количество посещений, тип, возможность заморозки).
- **Продажа абонементов:** Оформление и продажа абонементов спортсменам с автоматическим расчетом стоимости.

### 3. Система безопасности и контроля доступа
- **Ролевая модель:** Разграничение прав доступа для следующих ролей:
  - **Администратор:** Полный доступ ко всем функциям, включая администрирование системы и управление ценами.
  - **Тренер:** Доступ к расписанию и информации о своих группах.
  - **Бухгалтер:** Учет финансов, товаров, формирование отчетов.
  - **Продавец:** Базовые функции по работе с клиентами.
- **Авторизация:** Вход в систему по ФИО и паролю.

### 4. Финансовый модуль
- **Оплата услуг:** Поддержка различных форм оплаты (наличные, банковская карта, перевод).
- **Учет доходов и расходов:** Ведение учета по статьям, формирование отчета о прибылях и убытках.
- **Учет комиссии банка:** Автоматический расчет сумм поступлений с учетом эквайринга.

### 5. Операционный менеджмент
- **Расписание:** Визуальное планирование занятий для тренеров с учетом помещений.
- **Контроль посещений:** Регистрация входа спортсменов по карте с уведомлением об окончании абонемента.
- **Заморозка абонементов:** Управление приостановкой действия абонементов на срок до 3 месяцев.

### 6. Учет товаров (магазин при клубе)
- **Приход товара:** Оформление поступлений от поставщиков.
- **Продажа товара:** Регистрация розничных продаж с автоматическим расчетом цен и сумм.
- **Отчет по остаткам:** Контроль остатков товарно-материальных ценностей.

### 7. Отчетность
- **Отчет о продажах абонементов:** Анализ оказанных услуг по тренерам.
- **Отчет о продажах товаров:** Анализ розничных продаж.
- **Отчет по остаткам материалов:** Учет движения товаров.
- **Финансовый отчет:** Сводный отчет о доходах, расходах и прибыли за период.
